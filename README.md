## 🧐 About The Project

![wetube](./assets/images/screenshot.png)

Cloning Youtube with Vanilla and NodeJS

유저들은 영상 시청, 업로드, 자기 자신의 비디오를 녹화할 수 있고, 서로의 영상에 댓글을 달 수 있으며 영상을 검색할 수 있습니다.



### Built With

* MongoDB
* NodeJS
* ES6
* AWS
* Heroku
* SaSS
* WebPack
* Pug

## 😎 Getting Started

### Prerequisites

* npm

```sh
npm install
```

```sh
npm run dev:server
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/ooyuo/wetube.git
```

2. Install NPM packages

```sh
npm install
```
