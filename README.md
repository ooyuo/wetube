## 😎 Getting Started

### Prerequisites

* npm

```sh
npm install
```

```sh
npm run dev:server
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/ooyuo/wetube.git
```

2. Install NPM packages

```sh
npm install
```
